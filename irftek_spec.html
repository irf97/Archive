IrfTek Specification · Recursive Kernel · v0.1

Authority-Resolved Semantic Object Runtime

IrfTek is a local-first semantic runtime infrastructure where humans, devices, places, personas, tools, and AI agents expose structured state, coordinate locally, propagate through peers, render through simple UI, and anchor only what must become public or permanent.

1. Recursive Kernel

IrfTek = Object + Authority + Propagation + Projection + Anchor + Expiry

Object

The semantic thing: person, field, tool, device, session, persona, schema, dataset, AI agent, module.

Authority

Who owns truth for this object, in this context, for this action.

Propagation

How state moves: local bus, peer gossip, WebRTC, libp2p, IPFS snapshot.

Projection

How humans and machines see it: HTML/PWA, API, YAML, JSONL, SQLite, AI context.

Anchor

What becomes public, permanent, governed, or economically meaningful.

Expiry

What dies, compacts, summarizes, or becomes long-term memory.

Feature admission test

What object is this?
Who owns truth?
How does it spread?
How is it rendered?
What does AI read?
What expires?
What becomes permanent?
If a feature cannot reduce into this kernel, it is not IrfTek-structured yet.

9. AI-Readable Runtime

AI reads structured semantic objects, authority decisions, event streams, policies, memory summaries, and capability manifests. AI does not scrape opaque app state.

11. IMM / Machine Module Branch

IMM is the machine-authority branch under the IrfTek kernel, not the whole system.

IMM-S

Small edge node: ESP32-C3 / nRF52, BLE beacon, NFC optional, low-power identity and proximity events.

IMM-M

Local authority/runtime node: Raspberry Pi 5 / mini PC, Linux, Rust, SQLite, NATS/MQTT, Caddy, optional local AI.

IMM-X

Compute/cognition node: GPU box, Jetson, workstation, local model inference, semantic indexing, simulation.

13. Security + Governance

  • fake node

  • proximity spoof
  • malicious plugin
  • stolen wristband
  • poisoned schema
  • hostile peer flood
  • malicious authority
  • AI-generated false summary

Controls

  • Ed25519 signed events
  • NFC deliberate pairing
  • capability tokens
  • rate limiting
  • schema signatures
  • WASM sandbox
  • local revocation lists
  • content hashes
  • signed releases
